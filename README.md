# Lançamento Perfeito

Sistema pessoal para planejar, organizar e executar campanhas de marketing no modelo de lançamentos digitais.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma ORM
- SQLite em desenvolvimento

## Rodando localmente

```bash
npm install
npm run dev
```

O comando de desenvolvimento gera o Prisma Client, cria o banco SQLite local e roda o seed inicial quando o banco ainda está vazio.
